# Sistema Simples de Controle de Estoque

Sistema simples de controle de estoque desenvolvido com **HTML, CSS e JavaScript**, focado em organização e controle básico de produtos.

Projeto criado para fins de aprendizado e demonstração de lógica de programação para **TI Júnior**.

---

## Funcionalidades

- Cadastro de produtos
- Controle de quantidade
- Definição de estoque mínimo
- Alerta visual quando estoque está baixo
- Exclusão de produtos

---

##  Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

##  Como executar
Abra o arquivo `id2.html` no navegador
